HOW TO INSTALL MY APP
ON MY DEVICE
ACCOUNT CREATION

ONLY ONE TIME

APPLE ID

● To create your Apple ID, go to Apple Website

DEVELOPER ACCOUNT

● Choose an Apple Developer Program for Organizations or an Apple Developer Program for Individuals for App Store deployment
● Choose an Apple Developer Entreprise Program for in-house deployment

XCODE

● Go to the App Store, download and install Xcode

APPLE CONFIGURATOR 2

● Go to the App Store, download and install Apple configurator 2
XCODE CONFIGURATION

ONLY ONE TIME

ADD YOUR ACCOUNT

● Go to Xcode > Preferences > Accounts
● Add your account

TEAM ID

● Go here to get your Team ID
● Copy your team ID

4D FOR IOS

● Launch 4D for iOS
● Go to General Section and paste your Team ID reference in the Team field
APP INSTALL

EACH INSTALL

4D FOR IOS

● Go to BUILD tab
● Connect your device

INSTALL

● From the BUILD tab click on Install Button
● Wait for the app to be installed on your device
